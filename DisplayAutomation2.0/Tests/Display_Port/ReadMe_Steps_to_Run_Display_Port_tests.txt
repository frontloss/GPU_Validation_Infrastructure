Steps to run the DP tiled scripts:
---------------------------------

1. Download the latest DisplayAutomation2.0 binaries from Ubit server(http://ubit-gfxdispval.intel.com/overview/74).
2. Run 'setup.bat' on the Target machine from DisplayAutomation2.0 root. 
     -> This installs all required packages(like vcredist_x64.exe, python-2.7, DIVA etc.) to run the tests. 
3. Run 'env_setup.bat' on the Target machine from DisplayAutomation2.0 root to initialize the environment variables. 
4. Once the environment variables are set, Restart the machine.
4. Open command prompt in Administrative mode from DisplayAutomation2.0 root.
5. To run Display_Port Tiled test cases, Go to "DisplayAutomation2.0\Tests\Display_Port\DP_Tiled\TestSet_dp_tiled.xml".
     -> Open the dp_tiled.xml in a notepad(or some editor) to get the command line details.
     -> Copy-Paste the command line on command prompt and press Enter to run the test.
6. To run Display_Port SST test cases, Go to "DisplayAutomation2.0\Tests\Display_Port\DP_SST\TestSet_dp_sst_detection.xml".
     -> Open the dp_sst.xml in a notepad(or some editor) to get the command line details.
     -> Copy-Paste the command line on command prompt and press Enter to run the test.
7. The Logs generated by the test can be found under Logs folder: "DisplayAutomation2.0\Logs\".
     -> 'DisplayTest.log' will give the status of the test whether Pass/Fail.
     -> 'test_name.log' will be the detailed log of the test.
