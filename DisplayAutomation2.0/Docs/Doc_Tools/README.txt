Objective:
**********
This document breifs about the steps involved in generating the doxygen documentation for the test cases.

Step 1: Installation Steps:
	1.1. Install doxygen-1.811-setup.exe
	1.2. Install htmlhelp.exe in administrator mode

Step 2: Load Config File 
	2.1. C:\Program Files\doxygen\bin\doxywizard.exe
	2.2. Goto MainMenu-> File -> Open and select "PyGfxAutomation_Config" file

Step 3: Generate .CHM file
	3.1 Goto "Wizard" tab and update the source code directory path
	3.2 Goto "Expert" tab and "HTML" under "Topics" list menu
	3.3 Update output path in "CHM_FILE" item
	3.4 Goto "Run" tab and select "Run doxygen"
